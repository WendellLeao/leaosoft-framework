# Leaosoft Framework

A framework with many useful services and ready-to-use modular tools to make game development easier.

# How to install

## Via Git URL

Open the Unity Package Manager and add a new package from git URL using this link `https://github.com/WendellLeao/leaosoft-framework.git`

## Via OpenUPM

The package can be found by clicking the button below

[![openupm](https://img.shields.io/npm/v/com.leaosoft.gameplay.framework?label=openupm&registry_uri=https://package.openupm.com)](https://openupm.com/packages/com.leaosoft.gameplay.framework/)
